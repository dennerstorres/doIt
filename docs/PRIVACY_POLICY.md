# Política de Privacidade — doIt

Última atualização: 10 de Junho de 2026

A sua privacidade é importante para nós. Esta política descreve como o aplicativo **doIt** lida com suas informações.

## 1. Coleta de Informações

O **doIt** foi projetado para ser um aplicativo focado em privacidade e funcionamento offline.

- **Dados Pessoais:** Não solicitamos, coletamos ou armazenamos dados pessoais como nome, e-mail, número de telefone ou localização.
- **Conteúdo do Usuário:** Todas as tarefas, categorias, prazos e notas que você cria são armazenados exclusivamente no seu dispositivo.

## 2. Armazenamento e Segurança

- **Persistência Local:** Seus dados são salvos localmente usando o armazenamento do seu dispositivo.
- **Criptografia:** Implementamos criptografia AES-256 para proteger seus dados de tarefas enquanto estão em repouso no armazenamento local.
- **Sincronização:** Se você optar por configurar um endpoint de API personalizado para sincronização, seus dados serão transmitidos para esse endpoint. Nós não controlamos nem temos acesso aos dados enviados para servidores de terceiros configurados por você.

## 3. Permissões

O aplicativo solicita apenas as permissões estritamente necessárias:

- **Internet:** Necessária apenas se você utilizar a funcionalidade de sincronização remota opcional.

## 4. Compartilhamento de Dados

Não compartilhamos, vendemos ou alugamos seus dados para terceiros. Não utilizamos serviços de analytics ou rastreamento de terceiros que coletem dados de uso.

## 5. Alterações nesta Política

Podemos atualizar nossa Política de Privacidade ocasionalmente. Recomendamos que você revise esta página periodicamente para quaisquer alterações.

## 6. Contato

Se você tiver alguma dúvida sobre esta política, sinta-se à vontade para entrar em contato através do repositório oficial do projeto.
